Описание структуры программы:

1) Папка Input. В эту папку помещается файл выгрузки из 1C. Формат файла должен быть xls. Название файла не имеет значения.
2) Папка Output содержит в себе результат выполнения программы: два файла Nomenclature.xls и Ads.xls. 
	Nomenclature.xls - номенклатура разбитая на столбцы, 
	Ads.xls - готовые тексты объявлений
3) Папка Scripts содержит саму программу и логи. В ней ничего менять не нужно.
4) Папка Settings содержит файл с настройками appsettings.json.
5) Папка Template содержит макеты текстов для объявлений в формате txt. 

Описание настроек (appsettings.json):

1) NomenclatureHeaders содержит в себе список заголовков, которые находтся в выгрузке из 1C. 
	Так же этот список является списком заголовков в Файле Nomenclature.xls.
	Так же в нем находятся словари для нахождения нужных данных из номенклатуры из 1C.
2) AdsHeaders содержит в себе список заголовков, которые должны быть в Ads.xls.
	
Примеры работы с настройками:

1) Вы добавили новую колонку в выгрузку 1C, и она должна появиться в Nomenclature.xls. Заголовок добавляется в NomenclatureHeaders.
2) Вы добавили новую колонку в Nomenclature.xls и она должна появиться в Ads.xls. Заголовок добавляется в AdsHeaders.
3) Вы добавили новую колонку в выгрузку 1C и хотите добавить ее в Nomenclature.xls, но поменять название. 
	На пример в выгрузке 1C заголовок "Артикул", а в Nomenclature.xls должен быть "Артикул JORDAN". 
	Для этого в NomenclatureHeaders в заголовке "Артикул" добавляется подзаголовок "Артикул JORDAN" следующим образом:
	
	{
		"NomenclatureHeaders": {
			"Артикул": {
				"Артикул JORDAN": {}
			},
			...другие заголовки...
		}
	}
	(то же самое для смены имени заголовка в Ads.xls)
4) Вы добавили новые данные внутри текста "Номенклатура, Характеристика, Упаковка" выгрузки из 1C и хотите их достать оттуда, записав в новую колонку.
	Для этого нужно добавить новый подзаголовок в "Номенклатура, Характеристика, Упаковка", указав в нем текст для поиска, аналогично другим подзаголовкам. 
	Если рассмотреть подзаголовок "Подгруппа" 
	"Подгруппа": {
        "лев.": [ "LH" ],
        "прав.": [ "RH" ]
      },
	то текст для поиска располагается в квадратных скобках ["LH"], ["RH"]. А в кавычках "лев.", "прав." находится значение, в которое этот текст преобразуется.
	В квадратных скобках может быть несколько текстов для поиска: ["LH", "lh", "Lh", "lH"]. Учтите что поиск регистрозависимый. 
	Пример множественного поиска с заменой хорошо виден в подзаголовке "Марка", где марка автомобиля определяется по его модели.
	
Описание работы Templates (макеты текста объявлений):

1) В папке Templates содержаться txt файлы с макетами текстов объявлений. 
2) Название файла это название колонки в Ads.xls (название колонки с текстом не нужно прописывать в файле настроек. Колонка добавляется автоматически)
3) В txt файле находится текст объявления. В квадратных скобках находятся названия колонок Nomenclature.xls, текст из которых используется для подстановки.
4) Файлы можно добавлять, изменять, изменять названия, удалять.

Как работать с программой:

1) Загружаете xls файлы выгрузки из 1C в папку Input.
2) Запускаете Run.cmd двойным нажатием.
3) Забираете готовый результат в папке Output.
